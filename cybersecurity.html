Cybersecurity Interview Questions

February 13, 2019

Can you explain how confidentiality, integrity, and availability relate to software security vulnerabilities? What is the same origin policy? What does it protect against in web browsers? What is cross-site scripting (XSS)? Why is it useful for attackers? How can a web application defend against it? What is cross-site request forgery (CSRF)? Why is it useful for attackers? How can a web application defend against it? Are black lists or white lists preferential? How can a security team help developers write secure code? What kinds of security activities can a security team undertake with a development team? What are the pros and cons of each? (ex: penetration testing, threat modeling, etc..)
